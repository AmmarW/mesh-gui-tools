Detailed Description

A class for generating adaptive volume meshes.

This class provides functionality to generate a volume mesh by computing the Boolean difference between a cube and an inner mesh. The resulting mesh can be saved to a file.

Generates the volume mesh by computing the Boolean difference (cube \ inner).

Generates a volume mesh by computing the Boolean difference between a cube and an inner mesh.

Parameters
innerMeshFilePath to the inner mesh OFF file.
cubeSizeHalf-size of the cube (cube extends from -cubeSize to cubeSize).
outputMeshFileFilename for the output mesh (default is "adaptive_mesh.off").
Returns
Returns true if the mesh is generated successfully.

This function reads an inner mesh from a file, creates a cube of specified size, computes the Boolean difference between the cube and the inner mesh, and generates a volume mesh based on the difference. The resulting mesh is saved to an output file.

Parameters
innerMeshFileThe file path to the inner mesh.
cubeSizeThe size of the cube (bounding volume) to be created.
outputMeshFileThe file path where the output mesh will be saved.
Returns
True if the volume mesh generation is successful, false otherwise.
