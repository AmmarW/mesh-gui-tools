Detailed Description

Defines the MeshConverter class for converting between different mesh formats.

This file provides the MeshConverter class, which includes functions for converting OBJ files to OFF files, OFF files to OBJ files, and converting a custom Mesh object into a CGAL Polyhedron representation.
