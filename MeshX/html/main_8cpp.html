Detailed Description

Mesh viewer and editor application with ImGui-based GUI.

This application allows users to load, visualize, transform, and validate 3D meshes. It includes support for OBJ file import, boolean operations, adaptive mesh generation, and metadata assignment to mesh groups.

◆ main()

int main ( int argc,
char ** argv )

Entry point of the application.

Initializes GLFW and creates a window for rendering. Sets up ImGui context and initializes ImGui for OpenGL and GLFW. The main loop handles rendering, user input, and GUI interactions.

Parameters
argcNumber of command-line arguments.
argvArray of command-line arguments.
Returns
int Exit status of the application.

The main loop includes:

  • Camera controls for zoom and offset.
  • Adding and clearing OBJ files.
  • Displaying a list of loaded meshes.
  • Applying transformations to the active mesh.
  • Performing boolean operations on meshes.
  • Generating and exporting adaptive volume meshes.
  • Assigning metadata to mesh groups.
  • Logging application events and errors.

The rendering section uses OpenGL to draw the loaded meshes and handles camera transformations. ImGui windows are used for various controls and settings.
